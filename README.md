# Desafio Galeria de Fotos
Olá, esse desafio foi desenvolvido utilizando as tecnologias HTML, CSS E Javascript. Trata-se de uma galeria de fotos na qual a primeira vista é possível visualizar todas as imagens presentes, mas também possibilita pesquisar uma imagem que esteja presente na galeria, basta escrever o nome dela na barra de pesquisa. Se ela não estiver presente, o usuário será avisado que nenhuma foto foi encontrada.
